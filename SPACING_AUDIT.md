# Аудит расстояний между блоками

## Текущие значения:

### HUD:
- gap-2.5 (10px) - между TON/FISH и погодой ✓
- gap-2.5 (10px) - между TON и FISH ✓
- gap-2 (8px) - внутри чипов (иконка-текст) ✓

### BottomPanel:
- p-3.5 (14px) - внешний padding ✓
- p-3.5 (14px) - внутренний padding карточки ✓
- gap-2.5 (10px) - между элементами ✓
- mb-2.5 (10px) - между заголовком и кнопкой ✓

### BottomTabs:
- gap-1.5 (6px) - между табами ✓
- p-2 (8px) - внутренний padding ✓
- gap-0.5 (2px) - между иконкой и текстом ✓

### Страницы (Shop, Market, Tasks, Friends):
- p-3.5 (14px) - внешний padding страницы ✓
- gap-2.5 (10px) - между карточками в списке ✓
- gap-2 (8px) - между элементами внутри карточек ✓
- gap-3 (12px) - между иконкой и контентом в карточке удочки ⚠️
- mt-1.5 mb-2.5 - для заголовка ✓
- mt-2.5 - для некоторых элементов ⚠️
- mt-3 - для некоторых карточек ⚠️

## Проблемы:
1. gap-3 (12px) в карточках удочек не соответствует gap-2.5 (10px) в списках
2. mt-2.5 vs mt-3 - несоответствие
3. gap-2 vs gap-2.5 - нужно унифицировать

## Рекомендации:
- Унифицировать gap между карточками: gap-2.5 везде
- Унифицировать отступы между секциями: mt-2.5 везде
- Внутри карточек: gap-2 для мелких элементов, gap-2.5 для основных
