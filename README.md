# Ton Fish — Fishing Game

Современная игра-рыбалка для Telegram Mini Apps с полным игровым циклом. Зарабатывайте TON и FISH, ловите рыбу и развивайте свою коллекцию удочек!

## Технологии

- **React 18** + **TypeScript**
- **Vite** — быстрая сборка
- **TailwindCSS** — стилизация
- **React Router** — навигация
- **Zustand** — управление состоянием с persist

## Структура проекта

```
src/
├── components/     # React компоненты
├── pages/          # Страницы приложения
├── store/          # Zustand store
├── types/          # TypeScript типы
├── utils/          # Утилиты
├── data/           # JSON данные (rods, lootTables)
└── constants/      # Константы
```

## Запуск

```bash
npm install
npm run dev
```

Приложение откроется на `http://localhost:3000`

## Сборка

```bash
npm run build
```

## Игровой цикл

1. **Магазин** (`/shop`) — покупка и экипировка удочек
2. **Озеро** (`/lake`) — запуск 24-часовой сессии рыбалки
3. **Рыбалка** (`/fishing`) — активная сессия с таймером
4. **Улов** — модальное окно с результатом
5. **Рынок** (`/market`) — продажа улова
6. **Таски** (`/tasks`) — выполнение заданий
7. **Друзья** (`/friends`) — реферальная система

## Особенности

- ✅ Полный игровой цикл без бэкенда
- ✅ Сохранение состояния в `localStorage`
- ✅ Анимации и эффекты (glassmorphism, ripple, bob)
- ✅ Адаптивный дизайн для мобильных устройств
- ✅ Dev tools (5 тапов по балансу для ресурсов)

## Удочки

- **Палка** — за FISH (500 FISH)
- **Камышовая** — 0.1–25 TON, 2%/сутки
- **Бамбуковая** — 25–50 TON, 2.2%/сутки
- **Телескопическая** — 50–100 TON, 2.4%/сутки
- **Спиннинг** — 100–180 TON, 2.6%/сутки
- **Фидерная** — 180–280 TON, 2.8%/сутки
- **Boom Bait** — 280–560 TON, 3%/сутки
